# yarn-plugin-spdx

Create an SPDX SBOM file for the current project.

## Installation
```sh
yarn plugin import https://raw.githubusercontent.com/spdx/tools-ts/main/yarn-plugin/bundles/@yarnpkg/plugin-spdx.js
```

## Usage
```sh
yarn spdx --help
```
